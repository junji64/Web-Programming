HTML5 는 오디오 파일을 재생하기 위한 표준을 제공합니다

Audio on the Web

HTML5 이전에는 웹 페이지에서 오디오 파일을 재생하기위한 표준이 없었다..

HTML5 이전에는 오디오 파일을 플러그인(플래시 같은)으로 재생했으나, 서로 다른 브라우저는 서로 다른 플러그인을 지원했었다.

HTML5 는 웹 페이지에 오디오 파일을 포함하는 표준 방식을 지정하는 새로운 요소를 정의한다: <audio> 요소.

HTML5 Audio - How It Works

HTML5 에서 오디오 파일을 재생하려면 다음과 같이 한다:

Example

<audio controls>
  <source src="horse.ogg" type="audio/ogg">
  <source src="horse.mp3" type="audio/mpeg">
Your browser does not support the audio element.
</audio>

Try it yourself »

control 속성은 재생, 일시 정지, 볼륨과 같은 오디오 컨트롤을 추가합니다.

또한 <audio> 요소를 지원하지 않는 브라우저를 위해서 <audio> 및 </audio> 태그 사이에 텍스트 내용을 삽입합니다.

<audio> 요소에는 여러 <source> 요소가 올 수 있으며, <source> 요소는 서로 다른 오디오 파일에 링크 할 수 있습니다. 브라우저는 처음 으로 인식된 형식을 사용합니다.

Audio Formats and Browser Support

현재, <audio> 요소에서 지원되는 3 파일 형식: MP3, Wav, 과 Ogg:

Browser MP3 Wav Ogg
Internet Explorer YES NO NO
Chrome YES YES YES
Firefox NO
Update: Firefox 21 running on Windows 7, Windows 8, Windows Vista, and Android now supports MP3		 YES YES
Safari YES YES NO
Opera NO YES YES

MIME Types for Audio Formats

Format MIME-type
MP3 audio/mpeg
Ogg audio/ogg
Wav audio/wav

HTML5 Audio Tags

Tag Description
<audio> Defines sound content
<source> Defines multiple media resources for media elements, such as <video> and <audio>
