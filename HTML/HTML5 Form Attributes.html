HTML5 New Form Attributes

HTML5에서 <form> 및 <input>에 대한 몇 가지 새로운 속성을 가집니다.

<form> 에 새 속성:

  • autocomplete
  • novalidate

<input> 에 새 속성:

  • autocomplete
  • autofocus
  • form
  • formaction
  • formenctype
  • formmethod
  • formnovalidate
  • formtarget
  • height and width
  • list
  • min and max
  • multiple
  • pattern (regexp)
  • placeholder
  • required
  • step

<form> / <input> autocomplete 속 성

자동완성(autocomplete) 속성은 폼이나 입력 필드가 자동 완성 기능을 하는지 여부를 지정합니다.

자동완성 기능이 켜져있을 때, 브라우저는 사용자가 이전에 입력 한 값에 근거하여 값을 자동완성 한다.

Tip: form 에 대하여 자동 완성을 "on" 하고, 특정 입력 필드에는 그 기능을 "off" 할 수 있으며, 그 반대도 가능하다.

Note: 자동 완성 속성은 <form> 과 다음의 <input> type 에 사용한다 : text, search, url, tel, email, password, datepickers, range, and color.

Example

An HTML form with autocomplete on (and off for one input field):

<form action="demo_form.asp" autocomplete="on">
  First name:<input type="text" name="fname"><br>
  Last name: <input type="text" name="lname"><br>
  E-mail: <input type="email" name="email" autocomplete="off"><br>
  <input type="submit">
</form>

Try it yourself »

Tip: In some browsers you may need to activate the autocomplete function for this to work.

<form> novalidate Attribute

novalidate 속성은 부울(boolean) 속성이다.

존재하는 경우, 그것을 제출하면 양식 데이터 (input) 의 유효성을 검사하지 않도록 지정합니다.

Example

Indicates that the form is not to be validated on submit:

<form action="demo_form.asp" novalidate>
  E-mail: <input type="email" name="user_email">
  <input type="submit">
</form>

Try it yourself »

<input> autofocus 속성

autofocus 속성은 부울(boolean) 속성입니다.

<input> 요소가 자동으로 페이지를 로드 할 때 초점을 얻을 것을 지정합니다.

Example

Let the "First name" input field automatically get focus when the page loads:

First name:<input type="text" name="fname" autofocus>

Try it yourself »

<input> form 속성

form 속성은 <input> 요소가 속한 하나 이상의 form 을 지정합니다. 

Tip: 하나 이상의 form 을 참조할 경우, 공백으로 구분 된 form ID의 목록을 사용합니다.

Example

An input field located outside the HTML form (but still a part of the form):

<form action="demo_form.asp" id="form1">
  First name: <input type="text" name="fname"><br>
  <input type="submit" value="Submit">
</form>

Last name: <input type="text" name="lname" form="form1">

Try it yourself »

<input> formaction 속성

formaction 속성은 form 이 제출 될 때 입력 컨트롤을 처리하는 파일의 URL을 지정합니다. 

formaction 속성은 <form> 요소의 action 속성을 재설정합니다.

Note: formaction 속성은 type="submit" 와 type="image" 와 함께 사용됩니다.

Example

An HTML form with two submit buttons, with different actions:

<form action="demo_form.asp">
  First name: <input type="text" name="fname"><br>
  Last name: <input type="text" name="lname"><br>
  <input type="submit" value="Submit"><br>
  <input type="submit" formaction="demo_admin.asp"
  value="Submit as admin">
</form>

Try it yourself »

<input> formenctype 속성

formenctype 속성은 서버에 제출 될 때 폼 데이터(form-data)를 인코딩하는 방법을 지정합니다 (method="post" 인 forms 에서만 지원). 

formenctype 속성은 <form> 요소의  enctype 속성을 재설정한다.

Note: formenctype 속성은 type="submit" 와 type="image" 와 함께 사용됩니다.

Example

Send form-data that is default encoded (the first submit button), and encoded as "multipart/form-data" (the second submit button):

<form action="demo_post_enctype.asp" method="post">
  First name: <input type="text" name="fname"><br>
  <input type="submit" value="Submit">
  <input type="submit" formenctype="multipart/form-data"
  value="Submit as Multipart/form-data">
</form>

Try it yourself »

<input> formmethod 속성

formmethod 속성은 action URL에 폼 데이터를 보내기 위한 HTTP method를 정의합니다.

formmethod 속성은 <form> 요소의 method 속성을 재정의합니다.

Note: formmethod 속성은 type="submit" 와 type="image" 와 함께 사용됩니다.

Example

The second submit button overrides the HTTP method of the form:

<form action="demo_form.asp" method="get">
  First name: <input type="text" name="fname"><br>
  Last name: <input type="text" name="lname"><br>
  <input type="submit" value="Submit">
  <input type="submit" formmethod="post" formaction="demo_post.asp"
  value="Submit using POST">
</form>

Try it yourself »

<input> formnovalidate 속성

novalidate 속성은 부울(boolean) 속성입니다.

존재하는 경우, 그것을 제출하면 <input> 요소의 유효성을 검사하지 않도록 지정합니다.

formnovalidate  속성은 <form> 요소의 novalidate 속성을 재정의합니다.  

Note: formnovalidate 속성은 type="submit" 와 함께 사용됩니다.

Example

A form with two submit buttons (with and without validation):

<form action="demo_form.asp">
  E-mail: <input type="email" name="userid"><br>
  <input type="submit" value="Submit"><br>
  <input type="submit" formnovalidate value="Submit without validation">
</form>

Try it yourself »

<input> formtarget 속성

formtarget 속성은 양식을 제출 한 후 받은 응답을 표시 할 위치를 나타내는 이름 또는 키워드를 지정합니다.

formtarget 속성은 <form> 요소의 target 속성을 재정의합니다. 

Note: formtarget 속성은 type="submit" 와 type="image" 와 함께 사용됩니다.

Example

A form with two submit buttons, with different target windows:

<form action="demo_form.asp">
  First name: <input type="text" name="fname"><br>
  Last name: <input type="text" name="lname"><br>
  <input type="submit" value="Submit as normal">
  <input type="submit" formtarget="_blank"
  value="Submit to a new window">
</form>

Try it yourself »

<input> height 과 width 속성

height 과 width 속성은 <input> 요소의 높이와 너비를 지정합니다.

Note: height 과 width 속성은  <input type="image"> 와 만 사용됩니다.

Example

Define an image as the submit button, with height and width attributes:

<input type="image" src="img_submit.gif" alt="Submit" width="48" height="48">

Try it yourself »

<input> list 속성

list 속성은 <input> 요소에 대한 사전에 정의된 옵션이 포함 된 <datalist> 요소를 참조합니다.

Example

An <input> element with pre-defined values in a <datalist>:

<input list="browsers">

<datalist id="browsers">
  <option value="Internet Explorer">
  <option value="Firefox">
  <option value="Chrome">
  <option value="Opera">
  <option value="Safari">
</datalist>

Try it yourself »

<input> min and max 속성

min and max 속성은 <input> 요소에 대한 최소 및 최대 값을 지정한다.

Note: min 과 max 속성은 다음과 같은 input 형식들과 함께 사용된다: number, range, date, datetime, datetime-local, month, time and week.

Example

<input> elements with min and max values:

Enter a date before 1980-01-01:
<input type="date" name="bday" max="1979-12-31">

Enter a date after 2000-01-01:
<input type="date" name="bday" min="2000-01-02">

Quantity (between 1 and 5):
<input type="number" name="quantity" min="1" max="5">

Try it yourself »

<input> multiple 속성

multiple 속성은 부울(boolean) 속성입니다.

존재하는 경우, 그것을 제출하면 <input> 요소에 하나 이상의 값을 입력 할 수 있도록 지정된다. .

Note: multiple 속성은 다음과 같은 input 형식들과 함께 사용된다: email, and file.

Example

A file upload field that accepts multiple values:

Select images: <input type="file" name="img" multiple>

Try it yourself »

<input> pattern 속성

pattern 속성은 <input> 요소의 값에 대해 확인하는 정규 표현식(regular expression)을 지정합니다.

Note: pattern 속성은 다음과 같은 input 형식들과 함께 사용된다: text, search, url, tel, email, and password.

Tip: 사용자를 돕기위한 패턴을 설명하는데 global title 속성을 아용하시오. 

Tip: Learn more about regular expressions in our JavaScript tutorial.

Example

An input field that can contain only three letters (no numbers or special characters):

Country code: <input type="text" name="country_code" pattern="[A-Za-z]{3}" title="Three letter country code">

Try it yourself »

<input> placeholder 속성

placeholder 속성은 입력필드의 기대 값을 설명하는 짧은 힌트를 지정합니다(예를 들어, 샘플 값 또는 예상되는 형식의 간단한 설명).

짧은 힌트는 사용자가 값을 입력하기 전에 입력 필드에 표시된다.

Note: placeholder 속성은 다음과 같은 input 형식들과 함께 사용 된다: text, search, url, tel, email, and password.

Example

An input field with a placeholder text:

<input type="text" name="fname" placeholder="First name">

Try it yourself »

<input> required 속성

required 속성은 부울(boolean) 속성입니다.

존재하는 경우, 그 입력 필드가 양식을 제출하기 전에 작성되어야 함을 지정합니다. 

Note: required 속성은 다음과 같은 input 형식들과 함께 사용 된다: text, search, url, tel, email, password, date pickers, number, checkbox, radio, and file.

Example

A required input field:

Username: <input type="text" name="usrname" required>

Try it yourself »

<input> step 속성

step 속성은 <input> 요소의 합법적 번호 간격을 지정한다.

Example: step="3" 라면 , 합법적 번호는 -3, 0, 3, 6, 등 입니다.

Tip:  step 속성은 합법적인 값의 범위를 생성하기 위해 max 과 min 속성과 함께 사용될 수 있다.

Note:  step 속성은 다음과 같은 input 형식들과 함께 사용 된다: number, range, date, datetime, datetime-local, month, time and week.

Example

An input field with a specified legal number intervals:

<input type="number" name="points" step="3">
