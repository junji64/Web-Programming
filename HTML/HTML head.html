HTML <head> 요소

<head> 요소는 모든 헤드 요소를 담는 컨테이너다. <head> 내부의 요소들로는 스크립트와 브라우저에게 스타일 시트가 어디에 있는지를 알려주는 것, 메타정보제공 등이 올 수 있다.

다음 태그들이 head 섹션에 올 수 있습니다.: <title>, <style>, <meta>, <link>, <script>, <noscript>, and <base>.

The HTML <title>요소

<title> 태그는 문서의 제목을 정의한다.

<title> 요소는 모든 HTML / XHTML 문서에서 요구됩니다.

<title> 요소:

  • 브라우저 툴바의 제목을 정의
  • 즐겨 찾기에 추가 될 때 페이지 제목을 제공합니다
  • 검색 엔진의 검색 결과에서 페이지의 제목을 표시합니다

간단한 HTML 문서:

<!DOCTYPE html>
<html>
<head>
<title>Title of the document</title>
</head>

<body>
The content of the document......
</body>

</html>

<title> - Define a title for an HTML document
Use the <title> tag to define a title for a document.

HTML <base> 요소

<base> 태그는 페이지의 모든 상대적인 URL에 대한 기본 URL/대상을 지정합니다 :

<head>
<base href="http://www.w3schools.com/images/" target="_blank">
</head>

<base> - Default URL and target for all links
Use the <base> tag to specify a default URL and a default target for all links on a page.

HTML <link> 요소

<link> 태그는 현재 문서와 외부 자원 사이의 관계를 정의한다..

<link> 태그는 대부분 스타일 시트를 연결하는 데 사용됩 니다 :

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

HTML <style> 요소

<style> 태그는 HTML 문서에 대한 스타일 정보를 정의하는 데 사용됩니다.

<style> 요소 안에 HTML 요소들이 브라우저에 렌더링되는 방법들을 지정합니다:

<head>
<style type="text/css">
body {background-color:yellow;}
p {color:blue;}
</style>
</head>

HTML <meta> 요소

메타 데이터는 데이터에 관한 데이터 (정보)입니다.

<meta> 태그는 HTML 문서에 대한 메타 데이터를 제공합니다.  메타데이타는 페이지에 표시되지는 않지만, 기계에 의해 분석되어진다.

메타 요소는 일반적으로 페이지의 설명, 키워드, 문서의 작성자, 마지막 수정 및 기타 메타 데이터를 지정하는 데 사용됩니다.

메타 데이터는 브라우저에서 (콘텐츠 또는 다시로드 페이지를 표시하는 방법), 검색 엔진에서 (키워드), 또는 다른 웹 서비스에서 사용할 수 있습니다.

<meta> 태그는 항상 <head> 요 소 안에만 올 수있다

<meta> 태그 - 사용의 예

검색 엔진에 키워드를 정의합니다:

<meta name="keywords" content="HTML, CSS, XML, XHTML, JavaScript">

웹 페이지에 대한 설명을 정의합니다:

<meta name="description" content="Free Web tutorials on HTML and CSS">

페이지의 작성자를 정의합니다 :

<meta name="author" content="Hege Refsnes">

문서를 30초 후 새로 고침:

<meta http-equiv="refresh" content="30">

<meta> - Provide metadata for an HTML document
Use <meta> elements to specify a description, keywords, author, and character set of a document.

HTML <script> 요소

<script> 태그 JavaScript와 같은 클라이언트 측 스크립트를 정의하는 데 사용된다.

HTML 헤드 요소

Tag Description
<head> 문서에 대한 정보를 정의
<title> 문서의 제목을 정의합니다
<base> 페이지의 모든 링크에 대한 기본 주소 나 기본 대상을 정의
<link> 문서 와 외부 자원 사이의 관계를 정의
<meta> HTML 문서에 대한 메타 데이터를 정의
<script> 클라이언트 측 스크립트를 정의
<style> 문서의 스타일 정보를 정의
