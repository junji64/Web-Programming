iframe는 웹 페이지 내에서 웹 페이지를 표시하기 위해 사용된다.

iframe 을 추가하는 문법:

<iframe src="URL"></iframe>

src 속성은 iframe 페이지의 URL 을 명시한다.

Iframe - 높이 및 너비 설정

height 와 width 속성은 iframe 의 높이와 너비를 지정하는 데 사용됩니다..

속성 값은 기본적으로 픽셀 단위로 지정된다, 그러나 ( "80 %"처럼) %로도 할 수있다.

Example

<iframe src="demo_iframe.htm" width="200" height="200"></iframe>

Iframe - 테두리 제거

frameborder 속성은 iframe 주위에 테두리를 표시할지 여부를 지정합니다. 테두리를 제거하기 위해 속성 값 "0" 을 설정합니다:

Example

<iframe src="demo_iframe.htm" style="border:none"></iframe>
Try it Yourself »

링크의 타겟(target)으로 iframe 사용

iframe 은 링크에 대한 타겟 프레임으로서 사용될 수 있다

링크의 target 속성은 iframe 대응의 name 속성을 참조해야합니다:

Example

<iframe src="demo_iframe.htm" style="border:5px dotted red"></iframe>
