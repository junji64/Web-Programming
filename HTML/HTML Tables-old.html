HTML Tables

표는 <table> 태그로 정의된다.

표는 <tr> 태그로 정의되는 행(row)으 로 나누어진다. (tr 은 table row 를 의미한다)

행은 <td> 태그에 의해 데이터 셀로 나누어진다. (td 는 table data 를 의미한다)

또한, 행은 <th> 태그에 의해 제목들로 나누어질 수 도 있다. (th 는 table heading 을 의미한다)

<td> 요소는 표의 데이터를 담는다.

<td> 요소는 텍스트, 이미지, 목록, 다른 테이블 등과 같은 HTML 요소의 모든 종류를 포함 할 수 있습니다.

테이블의 폭은 CSS를 사용하여 정의 될 수 있다.

Example

<table style="width:300px">
<tr>
  <td>Jill</td>
  <td>Smith</td>
  <td>50</td>
</tr>
<tr>
  <td>Eve</td>
  <td>Jackson</td>
  <td>94</td>
</tr>
</table>

Border 속성을 가진 HTML Table

border 스타일을 지정하지 않을 경우, 테이블은 테두리 없이 표시됩니다.

테두리는 border 속성을 사용하여 추가할 수 있다.:

Example

<table border="1" style="width:300px">
<tr>
  <td>Jill</td>
  <td>Smith</td>
  <td>50</td>
</tr>
<tr>
  <td>Eve</td>
  <td>Jackson</td>
  <td>94</td>
</tr>
</table>

Try it yourself »

border 속성은 CSS를 사용하는 것이 바람직하다.

Example

<style>
table,th,td
{
border:1px solid black;
}
</style>

Try it yourself »

table 과 table cells 모두에 borders 속성을 정의해 야하는 것을 기억하시오.

접은 테두리의 HTML Table

테이블 테두리 와 표 셀 테두리가 하나의 테두리로 축소시키고 싶을경우, CSS에 border-collapse를 추가합니다.

Example

<style>
table,th,td
{
border-collapse:collapse
}
</style>

Try it yourself »

HTML Table 의 셀 패딩

padding을 지정하지 않을 경우, 테이블 셀은 패딩 없이 표시됩니다.

패딩을 갖는 테이블을 표시하려면, CSS에 padding을 사용합 니다:

Example

th,td
{
padding:5px;
}

Try it yourself »

HTML 표 제목

제목은 <th> 태그로 정의됩니다.

대부분의 브라우저는 제목을 굵은글씨로 중앙정렬되게 표시합니다.

Example

<table style="width:300px">
<tr>
  <th>First</th>
  <th>Last</th>
  <th>Points</th>
</tr>
<tr>
  <td>Eve</td>
  <td>Jackson</td>
  <td>94</td>
</tr>
</table>

Try it yourself »

제목을 왼쪽 정렬시키기 위해서는 CSS (text-align)를 사용한 다:

Example

th
{
text-align:left;
}

Try it yourself »

HTML 표의 셀 간격

셀 간격을 갖는 표를 표시하기 위해서는 CSS (border-spacing)를 사용한다. :

Example

table
{
border-spacing:5px;
}
