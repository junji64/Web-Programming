CSS로 HTML을 스타일링하기

CSS는 HTML 요소를 스타일링하는 더 나은 방법을 제공하기 위해서 HTML 4에서 도입되었다.

CSS는 다음과 같은 방법들을 HTML에 제공한다:

  • 인라인(Inline) - HTML 요소 안의 style 속성에 서
  • 내부 - <head> 섹션 안의 <style> 요소를 사용하여
  • 외부 - 외부의 CSS 파일을 사용하여

HTML에 CSS를 추가하는 선호되는 방법은, 외부의 별도 CSS 파일에 CSS 문법을 넣는 것이다.

하지만, 이 HTML 튜토리얼에서는 style 속성을 사용한 CSS를 소개한다. 이는 예를 단순화 하기 위한 것이며, 쉽게 코드를 편집하여 직접 실험을 쉽게 할 수 있게 해준다.

CSS에 대한 모든 것은 여기에서 배운다 ==> CSS Tutorial.

인라인(Inline) 스타일

독특한 스타일을 요소의 단 한번의 경우에만 적용 할 경우, 인라인 스타일이 사용될 수 있다.

인라인 스타일을 사용하려면, 해당 태그의 style 속성을 사용합니다. style 속성은 어떤 CSS 속성도 포함 할 수 있습니다. 아래의 예는 텍스트 색상 및 단락의 왼쪽 여백을 변경하는 방법을 보여줍니다:

<p style="color:blue;margin-left:20px;">This is a paragraph.</p>

HTML 스타일 Example - 배경 색

background-color 성질(property)은 요소의 배경색을 정의한다:

Example

<!DOCTYPE html>
<html>

<body style="background-color:yellow;">
<h2 style="background-color:red;">This is a heading</h2>
<p style="background-color:green;">This is a paragraph.</p>
</body>

</html>

HTML Style Example - 폰트, 색, 크기

font-family, color, 와 font-size 성질은  요소에 있는 텍스트의 폰트, 색과 크기를 정의한다:

Example

<!DOCTYPE html>
<html>

<body>
<h1 style="font-family:verdana;">A heading</h1>
<p style="font-family:arial;color:red;font-size:20px;">A paragraph.</p>
</body>

</html>

HTML Style Example - 텍스트 정렬

text-align 성질은 요소의 텍스트의 수평 정렬을 지정한다.:

Example

<!DOCTYPE html>
<html>

<body>
<h1 style="text-align:center;">Center-aligned heading</h1>
<p>This is a paragraph.</p>
</body>

</html>

내부 스타일 시트

단 하나의 문서에 고유한 스타일을 갖게 하려면 내부 스타일 시트를 사용할 수 있다. 내부 스타일 시트는 HTML 페이지의 <head> 섹션안의 다음과 같이 <style> 태그를 사용하여 정의된다.:

<head>
<style>
body {background-color:yellow;}
p {color:blue;}
</style>
</head>

외부 스타일 시트

스타일을 많은 페이지에 적용할 때 이상적이다. 외부 스타일 시트를 사용하면 하나의 파일을 변경하여 전체 웹 사이트의 모양을 변경할 수 있습니다.각 페이지는 <link> 태그를 사용하여 스타일 시트에 연결해야 합니다.  <link> 태그 는 <head> 섹션 안에 온다.:

<head>
<link rel="stylesheet" type="text/css" href="mystyle.css">
</head>

사용되지 않는 태그 및 속성

HTML 4 에서, 몇개의 태그와 속성이 문서를 스타일하는데 사용되었다. 이들 태그는 HTML 최신 버전에서는 지원되지 않는다.

사용하지 말아야 될 요소: <font>, <center>, 와 <strike>, 와 속성들: color 와 bgcolor.
