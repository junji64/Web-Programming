HTML 서식 요소

이전 장에서는 HTML 을 스타일링하는데 style 속성을 사용하는 방법을 배웠다.

HTML 에서는 텍스트에 특별한 의미를 부여하는 특별한 요소들이 있다.

HTML 에서는  <b> 와 <i> 같은 태그를 사용하여 출력서식을 굵게(bold)또는 기울임(italic) 꼴 텍스트로 한다.

포멧팅 요소들은 다음과 같이 텍스트를 특별한 타입으로 표시한다.: .

  • 굵은 텍스트(Bold text)
  • 중요 텍스트(Important text)
  • 기울임 꼴 텍스트(Italic text)
  • 강조 텍스트(Emphasized text)
  • 마크된 텍스트(Marked text)
  • 작은(Small) text
  • 삭제된(Deleted) text
  • 삽입된(Inserted) text
  • 아래첨자(Subscripts)
  • 위첨자(Superscripts)
대개의 경우 <strong> 은 <b> 로, <em> 은 <i>로 해석된다.

HTML 굵고(bold) 강한(strong) 서식

HTML <b> 요소는 별 다른 강조 없이 굵은(bold) 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><b>This text is bold</b>.</p>
Try it Yourself »
HTML <strong> 요소는 "강한(strong)" 의미를 추가한 강한 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><strong>This text is strong</strong>.</p>
Try it Yourself »

HTML 기울임(italic)과 강조(emphasized) 서식

HTML <i> 요소는 별 다른 강조 없이 기울임(italic) 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><i>This text is italic</i>.</p>
Try it Yourself »

HTML <em> 요소는 "강조(emphasis)" 의미를 추가한, 강조된(emphasized) 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><em>This text is emphasized</em>.</p>

HTML Deleted 서식

HTML <del> 요소는 삭제된 또는 제거된 텍스트를 정의한다. 

Example

<p>My favorite color is <del>blue</del> red.</p>

HTML 주석(comment) 태그

HTML 소스에 다음과 같은 문법으로 주석(comments)을 넣을 수 있다. 

Example

<!-- Write your comments here -->

주석은 브라우져에서 표시되지 않지만, 문서 정리에 도움을 준다.

주석을 사용하여 HTML에 알림 또는 다시 알림을 넣을 수 있다. 

HTML 의 여러 라인을 한 번에 주석 처리가 가능함으로, 오류 검색을 위한 디버깅(debugging)에 아주 좋다.

Example

Example

<!-- This is a comment -->

<p>This is a paragraph.</p>

<!-- Remember to add more information here -->

HTML 텍스트 서식 태그

Tag 설명
<b> 굵은 텍스트 정의
<em> 강조된 텍스트 정의
<i> 기울임 꼴 텍스트 정의
<small> 더 작은 텍스트 정의
<strong> 중요한 텍스트 정의
<sub> 아래 첨자 텍스트 정의
<sup> 윗 첨자 텍스트 정의
<ins> 첨가 텍스트 정의
<del> 지운 텍스트 정의
<mark> 마킹 / 강조된 텍스트 정의

HTML "컴퓨터 출력" 태그

Tag Description
<code> Defines computer code text
<kbd> Defines keyboard text 
<samp> Defines sample computer code
<var> Defines a variable

HTML 인용, 인용문, 그리고 정의 태그

Tag Description
<abbr> 약어(abbreviation) 및 머리 글자(acronym)를 정의
<address> 문서의 작성자 / 소유자에 대한 연락처 정보를 정의
<bdo> Defines the text direction
<blockquote> 다른 소스에서 인용되는 부분을 정의
<q> 인라인 (짧은) 인용을 정의
<cite> 작품의 제목을 정의합니다
<dfn> 정의 용어(definition term) 정의
