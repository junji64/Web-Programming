HTML 요소는 <div> 및 <span>를 갖고 그룹화 할 수 있다.

HTML 블록 요소

대부분의 HTML 요소는 블록 수준 요소로서 또는 인라인 요소로 정의된다.

블록 수준 요소는 일반적으로 브라우저에서 새 줄로 시작하거나 끝남으로 표시된다.

Examples: <h1>, <p>, <ul>, <table>

HTML 인라인 요소

인라인 요소는 일반적으로 새 줄을 시작하지 않고 표시됩니다.

Examples: <b>, <td>, <a>, <img>

HTML <div> 요소

HTML <div> 요소는 다른 HTML 요소를 그룹화하는 용기(Container)로서 사용될 수있는 블록 수준 요소이다.

<div> 요소는 특별한 의미는 없습니다.  이는 블록 수준 요소이므로 브라우저는 그 이전과 그 이후에 줄 바꿈을 표시합니다. 

CSS와 함께 사용하면, <div> 요소는 내용의 큰 블록에 속성 스타일을 설정하는 데 사용할 수있다.

<div> 요소의 또 다른 일반적인 사용은 문서의 배치(layout)입니다. 이는 테이블을 사용하여 배치를 정의하던 "옛날 방식"을 대체합니다. 배치를 위하여 <table> 요소를 사용하여 <table>의 올바른 사용이 아닙니다. <table> 요소의 목적은 테이블 형식의 데이터를 표시하는 것입니다.

HTML <span> 요소

HTML <span> 요소는 텍스트에 대한 용기(Container)로 사용할 수있는 인라인(inline) 요소입니다.

<span> 요소는 특별한 의미가 없습니다.

CSS와 함께 사용될 때, <span> 요소는 텍스트의 일정 부분에 속성 스타일을 설정하는 데 사용될 수있다.
