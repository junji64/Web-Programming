Check if your browser supports HTML5 video


Video on the Web

HTML5 이전에는, 웹 페이지에서 비디오/영화를 보여주는 표준이 없었다.

HTML5 이전에는, 동영상은 의 플러그인(flash 같은)으로만 보여질 수 있었으마, 다른 브라우저는 서로 다른 플러그인을 지원했었다.

HTML5 웹 페이지에 비디오나 동영상을 포함하는 표준 방식을 지정하는 새 요소를 정의합니다: <video> 요소.

 

HTML5 Video - How It Works

HTML5 에서 비디오를 표시하려면, 당신이 필요한 모두는 다음과 같다:

Example

<video width="320" height="240" controls>
  <source src="movie.mp4" type="video/mp4">
  <source src="movie.ogg" type="video/ogg">
Your browser does not support the video tag.
</video>

Try it yourself »

control 속성은 재생(play), 일시 정지(pause), 볼륨(volume)과 같은 비디오 컨트롤을 추가합니다.

또한 항상 width과 height 속성을 포함하는 것이 좋습니다. 높이와 폭이 모두 설정되면 페이지가 로드 될 때, 영상에 필요한 공간은 예약된다. 그러나, 이러한 속성이 없으면 브라우저는 비디오의 크기를 모르고, 그것에 적절한 공간을 확보 할 수 없다. 그 결과 비디오 로드하는 동안 페이지 레이아웃이 변경될 것이라는 것 입니다.

또한 <video>와 </video> 태그 사이에 <video> 요소를 지원하지 않는 브라우저를 위한 텍스트 내용을 삽입해야합니다.

<video> 요소는 여러 <source> 요소를 허용합니다. <source> 요소는 서로 다른 비디오 파일을 링크 할 수 있습니다. 브라우저는 처음 인식 되는 형식을 사용합니다.

Video Formats and Browser Support

현재 , <video> 요소에서는 3가지 비디오 포맷이 지원 됩니다: MP4, WebM, 과 Ogg:

Browser MP4 WebM Ogg
Internet Explorer YES NO NO
Chrome YES YES YES
Firefox NO
Update: Firefox 21 running on Windows 7, Windows 8, Windows Vista, and Android now supports MP4		 YES YES
Safari YES NO NO
Opera NO YES YES
  • MP4 = MPEG 4 files with H264 video codec and AAC audio codec
  • WebM = WebM files with VP8 video codec and Vorbis audio codec
  • Ogg = Ogg files with Theora video codec and Vorbis audio codec

MIME Types for Video Formats

Format MIME-type
MP4 video/mp4
WebM video/webm
Ogg video/ogg
