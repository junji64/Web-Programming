가장 일반적인 HTML 목록은 정렬(ordered) 목록과 비정렬(unordered) 목록이 있다:

HTML 목록

An ordered list:

  1. The first list item
  2. The second list item
  3. The third list item

An unordered list:

  • List item
  • List item
  • List item

HTML 비정렬(unordered) 목록

정렬되지 않은 목록은 <ul> 태그로 시작하며, 각 목록 항목은 <li> 태그로 시작된다.

목록 항목은 글 머리 기호 (일반적으로 작은 검은 색 원) 가 표시됩니다.

<ul>
<li>Coffee</li>
<li>Milk</li>
</ul>

위의 HTML 코드는 브라우저에서 이처럼 보여진다.

  • Coffee
  • Milk

HTML 정렬(ordered) 목록

정렬된 목록은 <ol> 태그로 시작되며.각 목록 항목은 <li> 태그로 시작된다.

목록 항목은 숫자로 표시됩니다.

<ol>
<li>Coffee</li>
<li>Milk</li>
</ol>

위의 HTML 코드는 브라우저에서 이처럼 보여진다.

  1. Coffee
  2. Milk

HTML 설명 목록

설명 목록은 설명과 함께 주어지는 각 용어/이름의 목록입니다.

<dl> 태그는 설명 목록을 정의 한다.

<dl> 태그는 <dt> (용어/이름 정의) 와 <dd> (용어/이름 설명)와 함께 사용된다.:

<dl>
<dt>Coffee</dt>
<dd>- black hot drink</dd>
<dt>Milk</dt>
<dd>- white cold drink</dd>
</dl>

위의 HTML 코드는 브라우저에서 이처럼 보여진다.

Coffee
- black hot drink
Milk
- white cold drink

HTML 목록 태그

Tag Description
<ol> 정렬 목록을 정의 
<ul> 비정렬 목록을 정믜 
<li> 항목을 정의 
<dl> 설명 목록을 정의 
<dt> 설명 목록에서 용어/이름을 정의
<dd> 설명 목록에서 용어/이름에 대한 설명을 정의
