HTML 제목

제목은 <h1> 부터 <h6> 태그들로 정의된다.

<h1> 은 가장 중요한 제목을 정의하며, <h6> 는 가장 덜 중요한 제목을 정의 한다.

Example

<h1>This is a heading</h1>
<h2>This is a heading</h2>
<h3>This is a heading</h3>
Try it yourself »

참고: 브라우저가 자동으로 각 제목 전/후에 약간의 빈 공간(여백)을 추가합니다.

제목은 중요합니다

HTML 제목은 제목으로만 사용하고,  텍스트를 BIG 또는 bold 로 만들기 위해 제목을 사용하지 마시오.

검색 엔진이 당신의 웹페이지의 구조와 내용을 색인하는데 제목을 사용합니다.

사용자들은 당신의 페이지를 제목으로 훑어보므로, 문서의 구조를 표시하는데 제목을 사용하는 것이 중요하다.

h1 제목을 주 제목으로, 그 다음에는 h2 제목을, 그리고 그 다음 덜 중요한 것에 h3 제목 등을 사용한다.

HTML 수평선(Horizontal Rules)

<hr> 태그는 HTML 페이지에 수평선을 생성합니다.

<hr> 요소는 내용을 시각적으로 구분할 때 사용될 수 있다:

Example

<p>This is a paragraph.</p>
<hr>
<p>This is a paragraph.</p>
<hr>
<p>This is a paragraph.</p>

HTML <head> 요소

HTML <head>요소는 HTML 제목(heading)과는 관련이 없다. 
HTML <head>요소는 meta data 를 담고 있으며,  meta data 는 보여지지 않는다.
HTML <head>요소는 <html> 태그와 <body> 태그 사이에 온다:

Example

<!DOCTYPE html>
<html>

<head>
  <title>My First HTML</title>
  <meta charset="UTF-8">
</head>

<body>

HTML Tip - HTML 소스 보는 방법

웹 페이지를 보다가,  "이걸 어떻게 했지?" 하며 궁금했던 적이 있습니까?

확인 하려면, 마우스 오른쪽 버튼을 클릭하고 "페이지 소스 보기(view page source)" 를 선택 또는 ctrl-U 를 입력합니다. 이로써, 페이지의 HTML 코드를 포함하는 창이 열립니다.
