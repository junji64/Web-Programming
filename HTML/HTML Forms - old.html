HTML forms 은 서로 다른 종류의 사용자 입력을 선택하는데 사용됩니다.


<form> 요소

HTML form 들은 사용자 입력을 수집하여 서버에 전달하는 데 사용됩니다.

<form> 요소로 HTML form 을 정의한다 :

<form>
.
form elements
.
</form>
HTML form 은 텍스트 필드, 체크 박스, 라디오 버튼, 제출 버튼 등과 같은 입력 요소를 포함 할 수 있으며,  또한 선택 목록,  텍스트 영역,  필드 셋,  범례, 및 레이블 요소를 포함 할 수 있다.

<input> 요소

가장 중요한 form 요소는 <input> 요소이다.

<input> 요소는 type 속성에 따라 여러 가지로 변화 될 수 있으며, 이 장에서는 간단히 다음과 같은 type 들을 사용해본다.:

Type Description
text 보통 텍스트 입력을 정의
radio 라디오 버튼 입력(다수의 경우 중 하나를 선택 )을 정의
submit 제출(form을 제출하는) 버튼을 정의

텍스트 입력 (text input)

<input type="text"> 사용자가 텍스트를 입력 할 수 있는 한 줄의 입력 필드를 정의한다 :

Example

<form>
First name:<br>
<input type="text" name="firstname">
<br>
Last name:<br>
<input type="text" name="lastname">
</form>

라디오 버튼

<input type="radio"> 라디오 버튼을 정의합니다. 라디오 버튼은 사용자가 제한된 개수의 선택 중 하나만 선택할 수 있습니다:

Example

<form>
<input type="radio" name="sex" value="male" checked>Male
<br>
<input type="radio" name="sex" value="female">Female
</form>

체크박스

<input type="checkbox"> 은 체크 박스를 정의합니다. 체크 박스는 사용자가 제한된 선택emf wnd의 0 개 이상의 옵션을 선택할 수 있습니다.defines a checkbox. Checkboxes let a user select ZERO or MORE options of a limited number of choices.

<form>

<input type="checkbox" name="vehicle" value="Car">I have a car

Submit Button

<input type="submit"> 는 폼-헨들러(form- handler) 에게 폼을 제출하는 버튼을 정의합니다.

폼-헨들러(form-handler)는 일반적으로 입력 자료를 처리할 스크립트로 이루어진 서버페이지 이다.

폼-헨들러(form-handler)는 폼의 action 속성에 명시되며 일반적으로 수신된 입력을 가지고 무언가를 수행합니다.:

Example

<form action="action_page.php">
First name:<br>
<input type="text" name="firstname" value="Mickey">
<br>
Last name:<br>
<input type="text" name="lastname" value="Mouse">
<br><br>
<input type="submit" value="Submit">
</form>
