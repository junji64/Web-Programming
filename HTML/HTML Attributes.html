속성은 HTML 요소에 추가 정보를 제공합니다.

HTML 속성

  • HTML 요소는 속성(attributes)들을 가질 수 있습니다.
  • 속성들은 요소에 대하여 추가적인 정보를 제공합니다.
  • 속성들은 항상 시작 태그에 명시됩니다.
  • 속성들은 이름(name)/값(value) 쌍으로 주어집니다 : name="value"

lang 속성

문서의 언어는 <html> 태그 안에 선언될 수 있으며, lang 속성으로 선언된다.

언어의 선언은 응용프로그램과 검색엔진의 접근성에 중요한다:

<!DOCTYPE html>
<html lang="en-US">
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
첫 두 문자(en)는 언어를 명시하며, 방언이 있다면, 두 문자(US)를 추가한다. 

title 속성

HTML 단락은 <p> 태그로 정의된다.

다음의 예에서, <p> 요소가 title 속성을 갖으며, 그 속성의 값은 "About W3Schools" 이다.

요소의 위로 마우스를 움직이면, title이 툴팁(tooltip)형태로 보여진다.:

Example

<p title="About W3Schools">
W3Schools is a web developer's site.
It provides tutorials and references covering
many aspects of web programming,
including HTML, CSS, JavaScript, XML, SQL, PHP, ASP, etc.
</p>

href 속성

HTML 링크는 <a> 태그로 정의 되며, 링크 주소는 href 속성에 지정된다:

Example

<a href="http://www.w3schools.com">This is a link</a>
Try it yourself »

size 속성들

HTML 이미지는 <img> 태그로 정의되며, 소스파일 이름(src), 이미지의 크기(width 와 height) 모두 속성으로 제공된다.:

Example

<img src="w3schools.jpg" width="104" height="142">
Try it Yourself »
위에서 alt 속성은 HTML요소가 표시될 수 없는 경우에 사용될 대체 텍스트를 명시한다.

항상 속성 값을 인용부호로 묶어주세요

속성 값은 항상 인용부호로 묶어주어야 합니다.

큰 따옴표(")가 가장 일반적이지만, 작은 따옴표(')가 사용될 수도 있습니다.

 Tip: 속성 값 자체가 따옴표를 포함하는 경우에는 작은 따옴표의 사용이 필요하기도 합니다: name='John "ShotGun" Nelson'

HTML Tip: 속성에 소문자 사용을 권장합니다.

속성 이름과 값은 대소문자를 구분하지 않습니다.

하지만 World Wide Web Consortium (W3C) 는 소문자 속성 이름/속성 값을 HTML 5 에서 권장하고 있습니다.

XHTML 에서는 소문자 속성이 요구됩니다.

HTML 속성 Reference

각 HTML 요소에 대한 속성의 전체 목록은 여기를 참조: HTML Tag Reference.

다음은 모든 HTML 요소에서 사용이 가능한 몇가지 속성의 목록입니다.:

속성 설명
id 요소에 대한 id를 명시
lang 요소에 사용된 언어의 선언
style 요소에 대한 inline CSS style 을 명시
title 요소에 대한 추가 정보를 제공 (tool tip의 형태로 보여짐)

For more information about global attributes: HTML Global Attributes Reference.
