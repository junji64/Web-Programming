<canvas> 요소는 웹 페이지에 실시간으로 그래픽을 그리는 데 사용됩니다.

왼쪽의 예는 캔버스에 그려진 빨간 사각형, 그라데이션(gradient) 사각형, 여러 가지 빛깔의 사각형, 및 여러 가지 빛깔의 텍스트를 보여줍니다.


What is Canvas?

HTML5 <canvas> 요소는 스크립트(보통 JavaScript)를 통해 즉석에서 그래픽을 그리는 데 사용됩니다.

The <canvas> 요소는 단지 그래픽을 위한 컨테이너입니다. 실제로 그래픽을 그리는데는 스크립트를 사용해야 합니다.

캔버스에는 경로, 상자, 원, 텍스트 그리기 및 이미지 추가를 위한 여러 가지 방법이 있습니다.

Create a Canvas

canvas 는 HTML 페이지에서 사각형 영역이며 HTML page, <canvas> 요소로 정의된다.

Note:<canvas> 는 기본적으로  테두리와 내용이 없습니다.

태그는 다음과 같다:

<canvas id="myCanvas" width="200" height="100"></canvas>

Note: 캔버스의 크기를 정의하기 위해서 항상 id 속성과 폭과 높이 속성을(스크립트에서 언급되는) 지정합니다 .

Tip: 하나의 HTML 페이지에 여러 <canvas>를 요소를 포함 할 수 있습니다.

테두리를 추가하려면, 스타일 속성을 사용:

Example

<canvas id="myCanvas" width="200" height="100"
style="border:1px solid #000000;">
</canvas>

JavaScript로 캔버스에 그리기

모든 캔버스에 그리기는 자바 스크립트 안에서 수행해야 합니다 :

Example

<script>
var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.fillStyle = "#FF0000";
ctx.fillRect(0,0,150,75);
</script>

Try it yourself »

Example explained:

먼저, <canvas> 요소를 찾습니다:

var c = document.getElementById("myCanvas");

그런 다음, getContext () 메소드를 호출합니다.( getContext() 메소드에 문자열 "2d"를 전달해야 합니다):

var ctx = c.getContext("2d");

getContext("2d") 객체는 내장된 HTML5 객체로써, 경로, 상자, 원, 텍스트, 이미지 등을 그리기 위한 많은 속성과 방법을 갖습니다:

ctx.fillStyle = "#FF0000";
ctx.fillRect(0,0,150,75);

fillStyle property 는 CSS 색상, 그라데이션 또는 패턴이 될 수 있으며, 기본 fillStyle 은 #000000 (블랙) 입니다.

fillRect(x,y,width,height) 메서드는 현재 fill style 으로 사각형을 그립니다.

캔버스 좌표

캔버스는 2 차원 격자입니다.

캔버스의 왼쪽-위 모서리가 (0, 0) 좌표입니다.

그래서, 위의 fillRect() 메소드는 매개 변수 (0,0,150,75)는 왼쪽-위 모서리 (0, 0)에서 시작하고 150x75 픽셀의 사각형을 그림을 의미합니다.

캔버스 - 경로

캔버스에 직선을 그리려면, 다음과 같은 두 가지 방법을 사용합니다 :

  • moveTo(x,y) : 라인의 시작점을 정의
  • lineTo(x,y) : 라인의 종료점을 정의

실제로 선을 그리려면, stroke() 과 같은 "ink" 메소드 중 하나를 사용해야 합니다

Example

위치(0, 0)에 시작 지점을 위치(200,100)에 끝 지점을 정의합니다.  그리고, stroke() 메서드를 사용하여 실제로 선을 그립니다:

Your browser does not support the HTML5 canvas tag.

JavaScript:

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.moveTo(0,0);
ctx.lineTo(200,100);
ctx.stroke();

Try it yourself »

캔버스에 원을 그리려면, 우리는 다음과 같은 메소드를 사용합니다 :

  • arc(x,y,r,start,stop)

실제로 원을 그리려면, stroke() 나 fill() 같은 "ink" methods를 사용합니다.

Example

Create a circle with the arc() method:

Your browser does not support the HTML5 canvas tag.

JavaScript:

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.beginPath();
ctx.arc(95,50,40,0,2*Math.PI);
ctx.stroke();

Canvas - Text

캔버스에 텍스트를 그리려면 가장 중요한 property성 및 methods은 다음과 같습니다 :

  • font - 텍스트의 font properties 을 정의
  • fillText(text,x,y) - 캔버스에 "채워진(filled)" 텍스트를 그립니다
  • strokeText(text,x,y) - 캔버스에 텍스트를 그립니다 (채우기 없음)

fillText() 사용:

Example

Write a 30px high filled text on the canvas, using the font "Arial":

Your browser does not support the HTML5 canvas tag.

JavaScript:

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.font = "30px Arial";
ctx.fillText("Hello World",10,50);

Try it yourself »

strokeText() 사용:

Example

Write a 30px high text (no fill) on the canvas, using the font "Arial":

Your browser does not support the HTML5 canvas tag.

JavaScript:

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
ctx.font = "30px Arial";
ctx.strokeText("Hello World",10,50);

Try it yourself »

Canvas - Gradients

Gradients 캔버스에 사각형, 원, 선, 텍스트 등을 채우기 위해 사용될 수있다. 캔버스에 도형은 단색에 한정되지 않는다.

그라디언트의 두 가지 종류가 있습니다:

  • createLinearGradient(x,y,x1,y1) - 선형 그라데이션을 만듭니다
  • createRadialGradient(x,y,r,x1,y1,r1) - 방사형 / 원형의 그라데이션을 만듭니다

그라데이션 객체에는 두 개 이상의 색상 멈춤 점을 추가해야합니다.

addColorStop() 메소드는 색상 정지 및 그라데이션을 따라 위치를 지정합니다. 그라데이션 위치는 0 에서 1 사이의 어디에나 있을 수 있습니다.

그라데이션을 사용하려면, 그라데이션 fillStyle 또는 strokeStyle property 정하고, 사각형, 텍스트 또는 라인과 같은 모양을 그립니다.

createLinearGradient() 사용하기:

Example

Create a linear gradient. Fill rectangle with the gradient:

Your browser does not support the HTML5 canvas tag.

JavaScript:

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");

// Create gradient
var grd = ctx.createLinearGradient(0,0,200,0);
grd.addColorStop(0,"red");
grd.addColorStop(1,"white");

// Fill with gradient
ctx.fillStyle = grd;
ctx.fillRect(10,10,150,80);

Try it yourself »

createRadialGradient() 사용하기:

Example

Create a radial/circular gradient. Fill rectangle with the gradient:

Your browser does not support the HTML5 canvas tag.

JavaScript:

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");

// Create gradient
var grd = ctx.createRadialGradient(75,50,5,90,60,100);
grd.addColorStop(0,"red");
grd.addColorStop(1,"white");

// Fill with gradient
ctx.fillStyle = grd;
ctx.fillRect(10,10,150,80);

Canvas - Images

캔버스에 이미지를 그리려면, 우리는 다음과 같은 method를 사용합니다 :

  • drawImage(image,x,y)

Image to use:

Example

Draw the image onto the canvas:

Your browser does not support the HTML5 canvas tag.

JavaScript:

var c = document.getElementById("myCanvas");
var ctx = c.getContext("2d");
var img = document.getElementById("scream");
ctx.drawImage(img,10,10);
