자바 스크립트(JavaScripts)는 HTML 페이지가 보다 동적이고 상호 작용을 하도록 해줍니다.

The HTML <script> Tag

<script> 태그는 JavaScript와 같은 클라이언트 측 스크립트를 정의하는 데 사용된다.

<script> 요소는 스크립트 문을 포함하거나, src 속성을 통해 외부 스크립트 파일을 연결한다.

자바 스크립트의 일반적인 용도는 이미지 조작, 양식 유효성 검사(form validation) 및 콘텐츠의 동적인 변화입니다.

아래의 스크립트는 HTML 출력으로 Hello World! 라고 쓴다.:

Example

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

The HTML <noscript> Tag

<noscript> 태그는 자신의 브라우저에서 스크립트 사용이 허용되지 않았거나 또는 클라이언트 측 스크립트를 지원하지 않는 브라우저를 가진 사용자에 대한 대체 콘텐츠를 제공하는 데 사용됩니다.

스크립트가 지원되지 않거나, 또는 사용자의 브라우저에서 허용되지 않은 경우는 <noscript> 요소 내부의 내용만 표시됩니다:

Example

<script>
document.getElementById("demo").innerHTML = "Hello JavaScript!";
</script>

<noscript>Sorry, your browser does not support JavaScript!</noscript>

자바 스크립트가 수행 할 수 있는 작업의 몇 가지 예는 다음과 같습니다 :

JavaScript can change HTML content:

document.getElementById("demo").innerHTML = "Hello JavaScript!";

JavaScript can change HTML styles:

document.getElementById("demo").style.fontSize = "25px";

JavaScript can change HTML attributes:

document.getElementById("image").src = "picture.gif";

HTML Script Tags

Tag Description
<script> Defines a client-side script
<noscript> Defines an alternate content for users that do not support client-side scripts
