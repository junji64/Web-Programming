HTML 서식 요소

이전 장에서는 HTML 을 스타일링하는데 style 속성을 사용하는 방법을 배웠다.

HTML 에서는 텍스트에 특별한 의미를 부여하는 특별한 요소들이 있다.

HTML 에서는  <b> 와 <i> 같은 태그를 사용하여 출력서식을 굵게(bold)또는 기 울임(italic) 꼴 텍스트로 한다.

포멧팅 요소들은 다음과 같이 텍스트를 특별한 타입으로 표시한다.: .

  • 굵은 텍스트(Bold text)
  • 중요 텍스트(Important text)
  • 기울임 꼴 텍스트(Italic text)
  • 강조 텍스트(Emphasized text)
  • 마크된 텍스트(Marked text)
  • 작은(Small) text
  • 삭제된(Deleted) text
  • 삽입된(Inserted) text
  • 아래첨자(Subscripts)
  • 위첨자(Superscripts)

HTML 굵고(bold) 강한(strong) 서식

HTML <b> 요소는 별 다른 강조 없이 굵은(bold) 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><b>This text is bold</b>.</p>

Try it Yourself »
HTML <strong> 요소는 "강한(strong)" 의미를 추가한 강한 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><strong>This text is strong</strong>.</p>

Try it Yourself »

HTML 기 울임(italic)과 강 조(emphasized) 서식

HTML <i> 요소는 별 다른 강조 없이 기울임(italic) 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><i>This text is italic</i>.</p>

Try it Yourself »

HTML <em> 요소는 "강조(emphasis)" 의미를 추가한, 강조된 (emphasized) 텍스트를 정의한다. 

Example

<p>This text is normal.</p>

<p><em>This text is emphasized</em>.</p>

Try it Yourself »

HTML Small 서식

HTML <small> 요소는 작은 텍스트를 정의한다. 

Example

<h2>HTML <small>Small</small> Formatting</h2>

HTML marked 서식

HTML <mark> 요소는 표시된(marked) 또는 하일라이티드(highlighted)텍스트를 정의한다. 

Example

<h2>HTML <mark>Marked</mark> Formatting</h2>

HTML Deleted 서식

HTML <del> 요소는 삭제된 또는 제 거된 텍스트를 정의한다. 

Example

<p>My favorite color is <del>blue</del> red.</p>

Try it Yourself »

HTML Inserted 서식

HTML <ins> 요소는 삽입된 또는 추 가된 텍스트를 정의한다. 

Example

<p>My favorite <ins>color</ins> is red.</p>

Try it Yourself »

HTML Subscript 서식

HTML <sub> 요소는 아래 첨자 텍스트를 정의한다. 

Example

<p>This is <sub>subscripted</sub> text.</p>

HTML Superscript 서식

HTML <sup> 요소는 위 첨자 텍스 트를 정의한다. 

Example

<p>This is <sup>superscripted</sup> text.</p>

HTML 텍스트 서식 태그

Tag 설명
<b> 굵은 텍스트 정의
<em> 강조된 텍스트 정의
<i> 기울임 꼴 텍스트 정의
<small> 더 작은 텍스트 정의
<strong> 중요한 텍스트 정의
<sub> 아래 첨자 텍스트 정의
<sup> 윗 첨자 텍스트 정의
<ins> 첨가 텍스트 정의
<del> 지운 텍스트 정의
<mark> 마킹 / 강조된 텍스트 정의

HTML "컴퓨터 출력" 태그

Tag Description
<code> Defines computer code text
<kbd> Defines keyboard text 
<samp> Defines sample computer code
<var> Defines a variable

HTML 인용, 인용문, 그리고 정의 태그

Tag Description
<abbr> 약어(abbreviation) 및 머리 글자(acronym)를 정의
<address> 문서의 작성자 / 소유자에 대한 연락처 정보를 정의
<bdo> Defines the text direction
<blockquote> 다른 소스에서 인용되는 부분을 정의
<q> 인라인 (짧은) 인용을 정의
<cite> 작품의 제목을 정의합니다
<dfn> 정의 용어(definition term) 정의
