HTML 문서는 HTML 요소(elemments)들로 만들어진다.

HTML 요소(Elements)

HTML 요소는 시작태그와 종료태그, 그리고 그 사이의 내용(content)으로 구성된다.

<tagname>content</tagname>
Start tag Element content End tag
<h1> My First Heading </h1>
<p> My first paragraph. </p>
<br>    

* 시작 태그는 종종 여는(opening) 태그, 종료 태그는 닫는(closing) 태그라 불린다.
* 어떤 HTML 요소는 종료(ending)태그를 갖지 않는다.

중첩된 HTML 요소

대부분의 HTML 요소는 중첩될 수 있다 (다른 HTML 요소를 포함할 수 있다).

다음의 HTML 문서는 중첩된 4개의 HTML 요소들로 구성된다.

Example

<!DOCTYPE html>
<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>
Try it yourself »

Example 설명

<html> 요소 는 전체 HTML 문서를 정의한다.
시작 태그 <html> 과 종료 태그 </html>를 갖으며,
요소의 내용은 또 다른 HTML 요소(<body> 요소) 이다.

<html>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

<body> 요소 는 HTML 문서의 본문을 정의한다.
요소는 시작 태그 <body> 와 종료 태그 </body>를 갖으며,
요소의 내용은 두개의 다른 HTML 요소( <h1> 과 <p>) 이다.

<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>

<h1> 요소는 제목(heading)을 정의한다.
요소는 시작 태그 <h1> 와 종료 태그 </h1>를 갖으며,
요소의 내용은 "My First Heading" 이다

<h1>My First Heading</h1>
<p> 요소 는 HTML 문서의 단락(paragraph)를 정의한다.
요소는 시작 태그 <p> 와 종료 태그 </p>를 갖으며,
요소의 내용은 "This is my first paragraph." 이다
<p>This is my first paragraph.</p>

종료 태그(End tag)를 잊지 마세요

일부 HTML 요소는 종료 태그를 잊은 경우에도 제대로 표시될 수 있습니다:

Example

<html>
<body>

<p>This is a paragraph
<p>This is a paragraph

</body>
</html>
Try it yourself »
대부분의 브라우져에서, 종료 태그는 선택사항으로 간주되기 때문에, 위의 예는 잘 작동됩니다.

하지만, 이것에 의존하지 마십시오. 종료 태그를 잊은 경우 대부분의 HTML 요소는 예기치 않은 결과/오류를 발생시킵니다.

빈(Empty) HTML 요소

내용이 없는 HTML 요소는 빈 요소라고 합니다.

<br> 은 종료 태그가 없는 빈 요소(<br> 태그는 줄 바꿈을 정의)입니다.

빈 요소들은 시작태그에서 다음과 같이 닫을 수도 있습니다: <br />

HTML5 는 빈 요소가 닫혀질 것을 요구하지는 않습니다.

HTML Tip: 소문자 태그를 사용을 권장합니다.

HTML 태그는 대소문자를 구분하지 않습니다: <P> 는 <p>와 같은 것을 의미한다. 많은 웹사이트는 대문자 HTML 태그를 사용한다.

World Wide Web Consortium (W3C)는 HTML 5에서 소문자를 요구하지는 않지만, XHTML에서는 소문자가 요구되므로 W3Schools 는 소문자를 사용합니다.
