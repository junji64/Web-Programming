HTML에서 예약된 문자는 문자 엔터티로 교체해야합니다.

키보드에 없는 문자 또한 문자 엔터티로 표시해야합니다.

HTML 엔터티

일부 문자는 HTML에 예약되어 있습니다.

당신이 텍스트에서 "보다 작다"(<) 기호와 "보다 크다" (>) 기호를 사용하는 경우, 브라우저가 태그와 혼합 할 수 있습니다.

문자 엔터티는 이러한 HTML에서 예약 된 문자를 표시하는 데 사용됩니다.

문자 엔티티는 다음과 같아 사용합니다.:

&entity_name;

OR

&#entity_number;

"보다 작다"(<) 기호를 표시하려면 &lt; 또는 &#60; 로 작성해야 합니다.

숫자 대신 엔터티 이름을 사용하는 것이 기억하기 쉬운 장점이 있습니다. 
단점은 브라우저가 모든 엔터티 이름을 지원하지 않을 수 있습니다, 하지만 숫자형은 모두 지원이 됩니다.

끊기지 않는 공백(Non Breaking Space)

HTML에 흔히 사용되는 문자 엔티티가 끊기지 않는 공백(non breaking space) (&nbsp;)이다.

브라우저는 항상 HTML 페이지에 공백들을 자른다는 것을 기억하십시오. 당신은 당신의 텍스트에 10개의 공백을 경우, 브라우저가 9 제거됩니다. 텍스트에 실제 공간을 추가하려면, 당신은 문자 엔티티 &nbsp; 를 사용할 수 있습니다.

HTML Symbols

HTML 기호 엔티티

수학 연산자, 화살표, 기술 기호 및 도형과 같은 HTML 기호는 일반 키보드에 존재하지 않습니다.

HTML 페이지에 다음 기호를 추가하려면, HTML 엔터티 이름을 사용할 수 있습니다.

엔터티 이름이 존재하지 않을 경우, 엔터티 번호를 사용할 수 있습니다.

문자 엔티티 이름이 없는 경우, 당신은 십진수 (또는 16 진수) 참조를 사용할 수 있습니다.


Example

<p>I will display &euro;</p>
<p>I will display &#8364;</p>
<p>I will display &#x20AC;</p>

Will display as:

I will display €
I will display €
I will display €
