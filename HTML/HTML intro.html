HTML 이란?

HTML은 웹 페이지를 기술하는 언어이다.

  • HTML은 Hyper Text Markup Language 를 의미한다.
  • HTML은 마크업 언어(markup language)을 사용하여 웹 페이지의 구조를 설명합니다.
  • HTML 요소는 HTML 페이지의 구성 요소입니다.
  • HTML 요소는 태그(tag)로 표현됩니다.
  • HTML 태그(tag)는 "heading", "paragraph", "table"등과 같은 내용의 레이블을 표시합니다.
  • 브라우저는 HTML 태그를 표시하지 않지만, 이를 사용하여 페이지의 내용을 렌더링합니다.

HTML Example

A small HTML document:

<!DOCTYPE html>
<html>
<head>
<title>Page Title</title>
</head>
<body>

<h1>My First Heading</h1>
<p>My first paragraph.</p>

</body>
</html>

Example 설명

  • DOCTYPE 선언은 문서의 형식을 정의 (HTML5에서의 문서타입 선언 방식)
  • <html>와 </html> 사이의 텍스트가 웹 페이지를 설명
  • <head>와 </head> 사이의 텍스트는 문서에 대한 정보를 제공한다.
  • <title>와 </title> 사이의 텍스트는 문서의 제목을 제공한다.
  • <body>와 </body> 사이의 텍스트가 표시되는 페이지 내용입니 다.
  • <h1>와 </h1> 사이의 텍스트는 제목으로 표시됩니다.
  • <p> 와 </p> 사이의 텍스트는 단락으로 표시됩니다.

HTML 태그(Tags)

HTML 마크업(markup) 태그는 보통 HTML 태그로 불린다.

  • HTML 태그는 <html> 처럼 꺽쇠괄호로 둘러싸인 키워드(태그 이름)이다.
  • HTML 태그는 일반적으로 <b> 와 </b> 처럼 쌍으로 주어진다.
  • 쌍의 첫 번째 태그가 시작 태그(start tag) 이고, 두 번째 태그가 종료 태그(end tag)이 다.
  • 종료 태그는 시작 태그와 같으나 그 이름 앞에 슬 래시(/)가 붙는다.
  • 시작 과 종료 태그는 때때로 열기 태그(opening tags) 와 닫기 태그(closing tags)로 불린다.
<tagname>content</tagname>

웹 브라우저(Web Browsers)

웹 브라우저들(Internet Explorer, Google Chrome, Firefox, Safari, 등)의 목적은 HTML 문서를 읽고 웹 페이지로 표시하는 것이다. (세계 각 나라들의 브라우저 선호도)

브라우저는 HTML 태그를 표시하지는 않지만, HTML 페이지가 사용자에게 어떻게 표시될/보여질 지를 결정하는데 태그를 사용한다.

HTML 페이지 구조

HTML 페이지 구조는 아래와 같이 묘사할 수 있다 :

<html>
<head>
<title>Page title</title>
</head>
<body>
<h1>This is a heading</h1>
<p>This is a paragraph.</p>
<p>This is another paragraph.</p>
</body>
</html>

 오직 <body> 영역(흰색 영역) 만이 브라우저로 보여진다.
