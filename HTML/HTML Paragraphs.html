HTML 문서는 단락들로 나누어집니다.

HTML 단락

단락은 <p> 태그로 정의된다.

Example

<p>This is a paragraph</p>
<p>This is another paragraph</p>
Try it yourself »

참고: 브라우저가 자동으로 단락의 전후에 빈 줄 하나를 추가한다.

HTML 표시

우리는 HTML 이 어떻게 표시될 지 확신 할 수 없다.

크거나 작은 스크린에서, 그리고 재조정된 윈도우에서 각기 다른 결과를 만들게 된다.

HTML에서는 코드에 여분의 공백이나 줄을 넣는다고해서 결과가 변하지 않는다. 브라우저는 페이지를 표시할 때 여분의 공백과 줄을 제거한다.  

어떤 개수의 공백과 어떤 개수의 줄바꿈도 오직 하나의 공백으로만 인식된다.

Example

<p>
This paragraph
contains a lot of lines
in the source code,
but the browser
ignores it.
</p>

<p>
This paragraph
contains         a lot of spaces
in the source         code,
but the        browser
ignores it.
</p>
Try it Yourself »

HTML 줄 바꿈

새 단락을 시작하지 않고, 줄 바꿈(새 줄)을 하려면 <br> 태그를 사용하시오:

Example

<p>This is<br>a para<br>graph with line breaks</p>
Try it yourself »

<br> 요소는 빈 HTML 요소로, 종료 태그가 없다.

HTML <pre> 요소

HTML <pre> 요소는 정해진 구조의 공백과 줄들로 이미-형식을 갖춘(pre-formatted) 텍스트의 블록을 정의한다. 

적절한 공백과 줄바꿈들로 표시하기 위해서는, 텍스트들을 <pre> 요소로 감싸주어야 한다:

Example

<pre>
  My Bonnie lies over the ocean.

  My Bonnie lies over the sea.

  My Bonnie lies over the ocean.

  Oh, bring back my Bonnie to me.
</pre>
