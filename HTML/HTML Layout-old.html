웹 페이지의 배치는 웹 사이트를 멋지게 보이게 하는데 매우 중요합니다.

웹 페이지의 배치는 매우 신중하게 디자인합니다.

웹 사이트 배치

대부분의 웹사이트는 내용을 여러 열/단 (잡지나 신문같은 형식)으로 보여준다.

여러 단(Multiple columns)은 <div> 나 <table> 요소를 사용해서 만들어진다. CSS 는 요소들을 위치시키거나, 페이지의 배경과 다채로운 모양을 만드는데 사용된다.

HTML 배치 - <div> 요소 사용

div 요소는 HTML 요소들을 그룹화하는데 사용되는 블록 수준의 요소이다.

다음 예는 이전 예에서와 마찬가지로, 여러 단의 배치를 생성하기 위해서 5개의 <div> 요소를 사용한다:

Example

<!DOCTYPE html>
<html>
<body>

<div id="container" style="width:500px">

<div id="header" style="background-color:#FFA500;">
<h1 style="margin-bottom:0;">Main Title of Web Page</h1></div>

<div id="menu" style="background-color:#FFD700;height:200px;width:100px;float:left;">
<b>Menu</b><br>
HTML<br>
CSS<br>
JavaScript</div>

<div id="content" style="background-color:#EEEEEE;height:200px;width:400px;float:left;">
Content goes here</div>

<div id="footer" style="background-color:#FFA500;clear:both;text-align:center;">
Copyright © W3Schools.com</div>

</div>

</body>
</html>

HTML Layout - 유용한 팁

Tip: CSS를 사용하는 가장 큰 장점은 외부 스타일 시트의 CSS 코드를 삽입하는 경우, 사이트를 유지하기 가 훨씬 쉽다는 것입니다. 하나의 파일만을 편집하여 모든 페이지의 레이아웃을 변경할 수 있게됩니다.

Tip: 고급 배치는 만드는데 시간이 걸리기 때문에, 빠른 옵션은 템플릿을 사용하는 것입니다. 무료 웹 사이트 템플릿 를 Google로 검색하시오 (사전 구축 된 웹 사이트의 레이아웃이며, 여려분이 수정해서 사용할 수 있습니다).
