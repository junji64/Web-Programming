HTML5 New Input Types

HTML5은 form에 대한 여러 가지 새로운 입력 유형이 있습니다. 이러한 새로운 기능은 더 나은 입력 제어와 유효성 검사를 할 수 있습니다.

새로운 입력 유형은 다음과 같습니다 :

  • color
  • date
  • datetime
  • datetime-local
  • email
  • month
  • number
  • range
  • search
  • tel
  • time
  • url
  • week

모든 브라우저가 새로운 입력 형식을 지원하지는 않으나, 당신은 사용을 시작할 수 있습니다; 지원이 되지 않는 경우엔, 일반 텍스트 필드처럼 작동합니다.

Input Type: color

color 형식 : 색상을 포함해야 하는 입력 필드에 사용됩니다.

Example

Select a color from a color picker:

Select your favorite color: <input type="color" name="favcolor">

Try it yourself »

Input Type: date

date type 사용자가 날짜를 선택할 수 있게 해줍니다.

Example

Define a date control:

Birthday: <input type="date" name="bday">

Try it yourself »

Input Type: datetime

datetime type 사용자가 (시간대와 함께) 날짜와 시간을 선택할 수 있게 해줍니다.

Example

Define a date and time control (with time zone):

Birthday (date and time): <input type="datetime" name="bdaytime">

Try it yourself »

Input Type: datetime-local

datetime-local type 은 사용자가 날짜와 시간 (시간대와는 무관하게)을 선택 할 수 있습니다.

Example

Define a date and time control (no time zone):

Birthday (date and time): <input type="datetime-local" name="bdaytime">

Try it yourself »

Input Type: email

email type e-메일 주소를 포함해야 하는 입력 필드에 사용된다.

Example

Define a field for an e-mail address (will be automatically validated when submitted):

E-mail: <input type="email" name="email">

Try it yourself »

Tip: iPhone 에서 Safari가 전자 메일 형식을 인식하고, 그에 따라서 on-screen 키보드를 변경 (@ 과 .com 을 추가).

Input Type: month

month type 사용자가 월 및 연도를 선택할 수 있습니다

Example

Define a month and year control (no time zone):

Birthday (month and year): <input type="month" name="bdaymonth">

Try it yourself »

Input Type: number

The number type숫자 값을 포함해야하는 입력 필드에 사용됩니다..

허용되는 번호에 대한 제한을 설정할 수 있습니다:

Example

Define a numeric field (with restrictions):

Quantity (between 1 and 5): <input type="number" name="quantity" min="1" max="5">

Try it yourself »

다음과 같은 속성을 사용하여 한계를 지정한다 :

  • max - 최대 허용값을 지정
  • min - 최소 허용값을 지정
  • step - 번호 간격 지정
  • value - 기본 값을 지정

모든 제한 속성의 예: Try it yourself

Input Type: range

range type 번호의 범위에서 값을 포함하는 입력 필드에 사용됩니다.

또한 어떤 번호가 허용되는지 제한 할 수 있습니다.

Example

Define a control for entering a number whose exact value is not important (like a slider control):

<input type="range" name="points" min="1" max="10">

Try it yourself »

제한을 지정하려면 다음과 같은 속성을 사용:

  • max - 허용된 최대 값을 지정
  • min - 허용된 최소값을 지정
  • step - 번호 간격 지정
  • value - 기본 값을 지정

Input Type: search

search type 검색 필드에 사용됩니다. (검색 필드는 일반 텍스트 필드처럼 동작)

Example

Define a search field (like a site search, or Google search):

Search Google: <input type="search" name="googlesearch">

Try it yourself »

Input Type: tel

tel type 전화 번호를 포함해야하는 입력 필드에 사용됩니다 

Example

Define a field for entering a telephone number:

Telephone: <input type="tel" name="usrtel">

Try it yourself »

Input Type: time

time type 사용자가 시간을 선택할 수게 해준다.

Example

Define a control for entering a time (no time zone):

Select a time: <input type="time" name="usr_time">

Try it yourself »

Input Type: url

url type 은 URL 주소를 포함하는 입력 필드에 사용됩니다.

폼이 제출 될 때 URL 필드의 값이 자동으로 확인됩니다.

Example

Define a field for entering a URL:

Add your homepage: <input type="url" name="homepage">

Try it yourself »

Tip: iPhone의 Safari는 URL 입력 방식을 인식하고,  맞추기위해서 on-screen 키보드를 변경합니다( .com 을 추가).

Input Type: week

week type 사용자가 주 및 연도를 선택할 수 있게 해준다.

Example

Define a week and year control (no time zone):

Select a week: <input type="week" name="week_year">

Try it yourself »
